##标签风暴

```
//默认配置
var _default = {
	//外容器选择器对象
	elem: '',
	//所有的标签(选择器对象)
	tags: [],
	//标签10个一组
	picLength: 15,
	//文字范围(圆的半径)
	R: 300,
	//自动轮播索引
	ii: 0,
	//随即大小
	randSize: ['10px', '14px', '16px', '18px', '24px', '26px', '28px', '32px'],
	//随即颜色
	randColor: ['#ed5450', '#e78419', '#329cb6', '#4894f6', '#598527', '#c6c6c6'],
	//默认动画效果由近及远(1)或由远及近(-1)
	torwards: -1,
	//是否自动切换(false或者空为不循环)
	auto: false,
	//定时器id
	autoId: null
};
```

###移动端支持左划右划
